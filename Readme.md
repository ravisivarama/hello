Pre-Requisites
1. Java 8 or higher
2. Maven (optional)

### Run the app

#### With Maven installed run 
`mvn spring-boot:run`

#### Without Maven Installed
For Unix use `./mvnw spring-boot:run`

For Windows `mvnw.cmd spring-boot:run`

Visit localhost:8383/hello or localhost:8383/hello/name-input-here
